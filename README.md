# 📊 Advanced Techniques in Data Visualization (Kenya Context)
## 📌 Project Overview
This project demonstrates advanced data visualization techniques using Python, focusing on real-world Kenyan datasets. It showcases how raw data can be transformed into meaningful insights through both static and interactive visualizations.
The project covers socio-economic analysis, healthcare trends, rainfall patterns, classification performance, and network relationships.
________________________________________
## 🛠️ Technologies Used
•	Python
•	Pandas
•	Seaborn
•	Matplotlib
•	Plotly
•	NetworkX
•	Anytree
________________________________________
## 📂 Features & Visualizations
### 1. 🔥 Correlation Heatmap
•	Analyzes relationships between socio-economic indicators such as:
o	Income
o	Education
o	Water Access
o	Electricity Access
o	Urbanization
•	Helps identify strong and weak correlations across counties.
________________________________________
### 2. 📉 Confusion Matrix Heatmap
•	Visualizes model performance in a construction compliance scenario.
•	Displays:
o	True Positives
o	False Positives
o	False Negatives
o	True Negatives
________________________________________
### 3. 🏥 Facet Grid (Healthcare Analysis)
•	Compares BMI vs Systolic Blood Pressure (SBP) across:
o	Counties
o	Gender
•	Enables multi-dimensional insights using small multiples.
________________________________________
### 4. 🌧️ Small Multiples (Rainfall Trends)
•	Displays monthly rainfall patterns across counties:
o	Nairobi
o	Kisumu
o	Mombasa
•	Useful for climate comparison and seasonal analysis.
________________________________________
### 5. 📈 Interactive Rainfall Dashboard (Plotly)
•	Interactive line chart with:
o	Hover insights
o	Multi-city comparison
o	Dynamic exploration
•	Enhances user engagement and storytelling.
________________________________________
### 6. 🌐 Network Graph (Logistics Hubs)
•	Models connections between major Kenyan cities:
o	Nairobi
o	Mombasa
o	Kisumu
o	Eldoret
o	Nakuru
•	Demonstrates graph-based visualization using NetworkX and Plotly.
________________________________________
###7. 🌳 Organizational Tree Structure
•	Represents a school hierarchy:
o	Principal → HODs → Teachers
•	Built using anytree for hierarchical visualization.
________________________________________
## 📊 Key Insights
•	Strong correlation exists between urbanization and infrastructure access.
•	Healthcare data shows positive trends between BMI and blood pressure.
•	Rainfall patterns vary significantly across regions.
•	Network graphs help visualize connectivity and central hubs (e.g., Nairobi).
